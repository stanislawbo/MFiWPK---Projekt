# MFiWPK---Projekt

Należy zaproponować język (możliwie prosty), w ramach którego użytkownik będzie mógł opisać formułę logiczną będącą koniunkcją wyrażeń logicznych. Użytkownik ma mieć możliwość:
• definiowania zmiennych (na potrzeby projektu nie trzeba zmiennych tablicowych, można po prostu pojedyncze zmienne),
• definiowania wyrażeń logicznych z koniunkcjami, implikacjami, alternatywami i negacjami.

Następnie, należy zaimplementować aplikację, która parsuje wejście od użytkownika w zaproponowanym języku i dokonuje przekształcenia wejściowej formuły do odpowiadającej formuły CNF. Dla przekształconej formuły CNF mamy wygenerować wejście w formacie DIMACS.
